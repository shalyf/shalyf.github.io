鼠标一般事件

鼠标跟踪事件

鼠标跟踪是通过定义一个矩形区域，捕获在这个区域内鼠标的进出和移动事件。这个监视区可以在awakeFromNib中用addTrackingArea:方法注册。

然而很多时候我们的View的frame或者bounds是会变化的，变化后在awakeFromNib中注册的监视区就与View不重合了，就会出现事件不会触发或者坐标偏差等现象。这个问题，我一般是通过重写updateTrackingAreas方法来解决，updateTrackingAreas在控件大小或者坐标发生变化的时候会被触发。

各个控件的拖拽实现

发表于 2015-11-13 | 分类于 macOS

NSOutlineView实现拖拽功能

注册拖拽类型

在OutlineView的AwakeFromNib中注册

public override void AwakeFromNib ()
{
    base.AwakeFromNib ();

    this.RegisterForDraggedTypes (new string[] {
        NSPasteboard.NSFilenamesType.ToString (),
        NSPasteboard.NSUrlType.ToString (),
        NSPasteboard.NSDragPasteboardName.ToString (),
        NSPasteboard.NSStringType.ToString (),
        NSPasteboard.NSFilesPromiseType.ToString ()
    });

    this.SetDraggingSourceOperationMask (NSDragOperation.All, false);
    this.AllowsMultipleSelection = true;
}

RegisterForDraggedTypes注册拖动类型，仅有注册的类型才能完成控件内的拖动（有拖动特效），如果是要拖动到控件以外的地方，则可以不用注册。
SetDraggingSourceOperationMask设置拖动源操作，没有设置的话将不会呼起必须的拖动方法，特别注意第二个参数，置为true表示拖动只能在控件内，置为false表示可以拖动到控件以外。如果需要拖动到桌面等其他文件夹，一定要置为false。

实现OutlineViewwriteItemstoPasteboard方法（在DataSource中）

public override bool OutlineViewwriteItemstoPasteboard (NSOutlineView outlineView, NSArray items, NSPasteboard pboard)
{
    pboard.DeclareTypes (new string[] {
    NSPasteboard.NSFilesPromiseType.ToString () }, null);
    string[] types = new string[1] { "" };
    NSArray list = NSArray.FromStrings (types);
    pboard.SetPropertyListForType (list, NSPasteboard.NSFilesPromiseType.ToString ());
    return true;
}

其中pboard.DeclareTypes声明拖拽类型。
pboard.SetStringForType或pboard.SetDataForType设置拖动的内容。案例使用SetPropertyListForType为粘贴板设置一个列表内容。
当拖动类型为NSPasteboard.NSFilenamesType时，list中存储的url必须是针对本机有效的，如果用了无效的url，将无法完成拖动。
当拖动类型为NSPasteboard.NSFilesPromiseType时，list可以随便填，但是不能为空，该方法还没有完全理透，不过已经可以通过这个方法来获取目标Url。使用这种拖动类型将会呼起FilesDropped方法，通过该方法就能获得目标Url。

实现DragImage方法（在控件中，需要重写控件，也可以不实现，使用父类方法）

public override void DragImage (NSImage anImage, PointF viewLocation, SizeF initialOffset, NSEvent theEvent, NSPasteboard pboard, NSObject sourceObj, bool slideFlag)
{
    anImage = NSWorkspace.SharedWorkspace.IconForFile ("/Applications/Tongbu.app");
    viewLocation = this.ConvertPointFromView (theEvent.LocationInWindow, null);
    viewLocation.X -= 16;
    viewLocation.Y -= 16;

    base.DragImage (anImage, viewLocation, initialOffset, theEvent, pboard, sourceObj, slideFlag);
}

anImage设置拖动时显示的图标，viewLocation设置图标显示的位置。
然后调用父类的DragImage方法完成。

实现FilesDropped方法（在DataSource中）

使用NSPasteboard.NSFilesPromiseType拖动类型时触发
如果拖动源不是本机有效的Url，使用NSPasteboard.NSFilenamesType是不允许拖动的，这时候就要用到NSPasteboard.NSFilesPromiseType了。

public override string[] FilesDropped (NSOutlineView outlineView, NSUrl dropDestination, NSArray items)
{
    Console.WriteLine ("dropDestination = " + dropDestination.Path);
    return new string[] { "" } ;
}

dropDestination为目标Url，获取该Url，就能通过其他手段将需要的文件复制到目的地了。

实现WriteRowsWithIndexesToPasteboard方法（在Delegate中）

public override bool WriteRowsWithIndexesToPasteboard (NSBrowser browser, NSIndexSet rowIndexes, int column, NSPasteboard pasteboard)
{
    string[] temp = new string[1]{""};
    NSArray fileList = NSArray.FromStrings (temp);
    pasteboard.DeclareTypes (new string[] { NSPasteboard.NSFilesPromiseType.ToString() }, null);
    pasteboard.SetPropertyListForType (fileList, NSPasteboard.NSFilesPromiseType.ToString());
    return true;
}
